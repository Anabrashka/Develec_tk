# Develec_tk

Proyecto desarrollado en Unity WebGL con integración de MQTT utilizando EMQX como broker.

## 📌 Descripción
Este repositorio contiene un proyecto de Unity que busca establecer comunicación en tiempo real utilizando el protocolo MQTT, ideal para aplicaciones IoT y sistemas interactivos basados en WebGL.

## 🚀 Tecnologías utilizadas
- **Unity** (versión recomendada: 2022.3 LTS o superior)
- **WebGL** (exportación desde Unity)
- **EMQX** (Broker MQTT)
- **MQTT.js** (para posibles pruebas en navegador)

## 🎯 Objetivos
- Configurar un cliente MQTT en Unity WebGL.
- Comunicar el cliente con un broker EMQX.
- Enviar y recibir mensajes en tiempo real.
- Documentar todo el flujo de trabajo y configuración.
